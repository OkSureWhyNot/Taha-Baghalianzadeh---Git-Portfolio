FTC DECODE 2025–2026 — Targeting & Shooting System

Overview

This repository contains a curated selection of code from our FIRST Tech Challenge robot's 2025–2026 DECODE season Steampunks Alpha 9040. I served as Mechanical Lead and Programmer on the team, and the files here represent the real-time targeting and shooting system I worked on: tracking the field goal, calculating the shot needed from the robot's current position even if the robot was moving, and driving the turret, hood, and flywheel to take that shot automatically.

This is not the full team codebase, just the files most relevant to that system.
